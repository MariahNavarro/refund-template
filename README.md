# 💸 Sistema de Reembolso

Este é um projeto simples para controle de despesas, onde o usuário pode adicionar gastos e visualizar o total de forma rápida.

## 📌 Sobre o projeto

O **Sistema de Reembolso** permite que o usuário:

- Adicione uma despesa informando:
  - Nome
  - Categoria
  - Valorr
- Visualize todas as despesas cadastradas
- Veja o total de despesas e o valor total
- Exclua uma despesa clicando no "X"

A ideia do projeto é facilitar o controle de gastos de forma simples e organizada.

## 🖥️ Interface

O projeto possui uma interface visual intuitiva, com:

- Formulário para adicionar despesas
- Lista das despesas cadastradas
- Exibição do total e quantidade de despesas
- Ícones para identificar cada categoria
- Botão para remover gastos

## 🚀 Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## ⚙️ Como usar

1. Abra o arquivo `index.html` no navegador  
2. Digite o nome dos gastos
3. Escolha as categorias
4. Informe o valor  
5. Clique em "Adicionar gasto"
6. Veja a despesa aparecer na lista ao lado  
7. Para excluir, clique no "X"  

## 📁 Estrutura do projeto

- `index.html` → Estrutura da página  
- `style.css` → Estilo visual  
- `script.js` → Lógica do sistema  

## 💡 Observação

Este projeto não salva os dados após atualizar a página. Para isso, seria necessário usar armazenamento local (LocalStorage) ou um banco de dados.
